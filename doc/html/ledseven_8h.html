struct pinOut PinOut

A typedef for the mapping of pins between the Arduino and seven- segment LED display.

This type is to be used when creating the mapping between the pins of the Arduino and those of the seven-segment LED display. For details on creating the mapping, see the documentation for pinOut.

Prints a decimal digit to the display.

The printDec function is designed to print a decimal digit to a seven-segment LED display.
